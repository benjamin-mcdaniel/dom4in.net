About this data

dom4in.net samples the short-domain space and reports high-level aggregates only. For the major TLDs tracked here, essentially all 1–3 character labels are already registered; this dashboard still samples those spaces but never lists individual domains, only aggregate behavior.

Possible labels is the theoretical size of the short-name space at that length — how many simple strings exist before the dot, under the character set the collector uses (letters a–z and digits 0–9, 36 options per character). Tracked checks counts how many domain checks have been run for labels of that length across a fixed set of TLDs (com, net, org, io, co, info, biz, online, store, app, dev, ai, xyz, me). A label may be checked on one TLD, several, or none, so this is the number of checks performed, not a complete labels × TLDs grid.

The bars above show what fraction of checked domains appear unregistered (no DNS) or registered but unused/parked. These percentages are computed from the same aggregates and never expose individual domains. WHOIS and registration data for specific domains are operated by each TLD registry or registrar; dom4in.net does not query WHOIS directly.

Stats API

Everything on this page comes from a single public JSON endpoint:

curl https://dom4in.net/api/stats/overview
Aggregates only. No individual domains are listed or stored.
