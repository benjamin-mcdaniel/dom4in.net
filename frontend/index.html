About this data

dom4in.net samples the short-domain space and reports high-level aggregates only. For the major TLDs we look at, essentially all 1–3 character labels are already registered; this dashboard still samples those spaces but never lists individual domains, only aggregate behavior.

Possible Labels is the theoretical size of the short-name space at that length – how many simple strings exist before the dot, under the character set the collector is using (currently letters a–z and digits 0–9, 36 options per character). In practice, real short domains may also contain other symbols such as hyphens; the length charts should be read as "characters in the label before the dot" regardless of whether those characters are letters or numbers. Tracked counts how many domain checks we have actually run so far for labels of that length across a fixed set of TLDs (com, net, org, io, co, info, biz, online, store, app, dev, ai, xyz, me). A given label may be checked on one TLD, several TLDs, or not yet checked at all, so Tracked is the number of domain-level checks performed, not a complete labels × TLDs grid.

Below, we also summarize what fraction of the checked short domains appear unregistered (no DNS) or registered but unused/parked. These percentages are computed from the same aggregates and never expose individual domains. WHOIS and registration data for specific domains are operated by each TLD registry or registrar; ICANN coordinates the system but does not run a central public WHOIS service, and dom4in.net does not query WHOIS directly.
