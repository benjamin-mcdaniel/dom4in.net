Possible Labels is the size of the short-name space at that length – how many letter strings exist (a–z) before the dot. For example, length 2 has 26² = 676 labels (aa, ab, …). Tracked counts how many domain searches we have actually run so far, across a fixed set of TLDs (com, net, org, io, co). Each label can be checked on several TLDs, so Tracked can be larger than Possible Labels, because it is counting labels × TLDs, not unique labels.

About this data

dom4in.net samples the short-domain space and reports high-level aggregates only. We currently focus on letter-based labels up to 6 characters and combine results across several major TLDs, so counts represent label coverage rather than a list of specific domains.
